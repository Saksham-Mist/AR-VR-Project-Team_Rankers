# AR-VR-Project-Team_Rankers
#This project is focused on making a Geoetric Transformation Environment for professionals and non-professionals to visually learn about 3D systems from the pre-existing 2D system of learning to combat errors in coordinate alignment and calibration.

The project mostly focuses on creating an environment for learning 3D based mkdels and coordinate systems using 3D softwares, a significant step up from the existing methodology of learning through 2D diagrams and matrices. This will, in the long term, benefit professionals as well as enthusiasts in terms of errors, profits in the long-term
