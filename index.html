📘 Project Overview

Traditional engineering and computer graphics education relies heavily on 2D diagrams to explain inherently 3D concepts such as rotation, translation, and composite transformations.

Our system replaces static diagrams with real-time 3D interaction. Users can manipulate objects while observing the transformation matrix update instantly, creating a direct link between mathematical representation and spatial movement.

How to Use the Demo:

  • Open the Live Demo link above
  • Use sliders to translate and rotate the cube
  • Observe real-time matrix updates
  • Use mouse to orbit around the object

This prototype validates the interaction model before full VR implementation.
